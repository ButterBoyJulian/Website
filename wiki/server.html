Julian Network Server

The Beginning

At first, a Minecraft server was just an idea to ButterBoyJulian, the owner of the server. Julian had the idea of making a server to connect many small servers that don't have a lot of players to create a mega-network. Over time this idea changed but its original thought is still present in the server.

The Alpha Era

The Julian Network Minecraft Server was created on Monday, October 16, 2017. In 2017, it was originally a factions server but because of a huge error, the owner accidentally reinstalled a backup that reverted the server to a blank world.

After that disaster, the server turned into what we now call Enchanted Survival. It had custom enchants and crates at spawn. This did not last however, the server was later turned into a Bungeecord network.

The Beta Era (Now)

The server remains to still be in its beta phase. This is because there is still a list of errors that need fixing on the server. After this, the server will be released and the owner will start getting as many players online the server as possible.
